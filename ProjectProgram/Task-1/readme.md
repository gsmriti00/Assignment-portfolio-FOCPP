Focus

Variables, numbers (floats), lists, loops, conditionals, and input/output.


Purpose

To collect pizza prices, apply a 4-for-3 offer, and display the final total with the discount.


Input Validation

Each pizza price must be a valid number greater than zero. The program keeps asking until correct input is entered.


Description

The program asks the user to enter the prices of four pizzas. All prices are stored in a list and the cheapest pizza is made free as part of the 4-for-3 offer. The program then calculates the total price to pay and works out the discount percentage.

The discount percentage applied
